# actrec-general

Contém projeto geral sobre reconhecimento/predição de ações/atividades humanas.

O objetivo central é dispor de um texto técnico atualizado que permita a redação de propostas para concorrência em editais de fomento de pesquisa, tais como pibic, universal, papespa, etc.

Arquivos:
- specific-aim-document: documento de resumo executivo ou de objetivos específicos; fornece um apanhado geral do projeto e estabelece diretrizes para redação do projeto geral
- main: arquivo principal do projeto geral
  - preambule: preâmbulo com cabeçário latex
